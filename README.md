## Projeto
Este projeto foi desenvolvido como trabalho acadêmico para a disciplina de Big Data, no curso de Sistemas de Informação da Universidade Veiga de Almeida (UVA).
O objetivo do projeto é analisar a quantidade mensal de queimadas, identificando padrões, destacando meses críticos e gerando relatórios e gráficos a partir dos dados.

## Sobre o Sistema
O sistema vai ler um arquivo .CSV com os dados das queimadas, esses dados será tratados para mudar seu formato, identificando meses críticos e aplicando alertas quando o limite é ultrapassado gerando saídas automáticas.
Ele vai produzir um gráfico com as mudanças mensais das queimadas e gera um relatórios com os meses que atingiram os níveis críticos.

## Objetivo
- Permitir a análise sazonal das queimadas durante o ano.
- Identificar automaticamente os meses mais críticos.
- Facilitar a visualização dos dados através de gráfico.
- Registrar os meses que ultrapassaram o limite de alerta, informando a gravidade dos focos.
